# Cluster Analysis using KMeans and Hierarchical Clustering

## 📌 Project Overview

This project performs **cluster analysis** using both:

* Hierarchical Clustering (Dendrogram)
* KMeans Clustering

The goal is to group similar data points into clusters using unsupervised learning techniques.

---

## ⚙️ Workflow

1. Load dataset from Excel
2. Data preprocessing
3. Perform Hierarchical Clustering
4. Visualize clusters using Dendrogram
5. Apply KMeans clustering
6. Assign cluster IDs
7. Evaluate clustering using:

   * Silhouette Score
   * Elbow Method

---

## 📊 Key Concepts

* Unsupervised Learning
* KMeans Algorithm
* Hierarchical Clustering
* Dendrogram Visualization
* Silhouette Analysis
* Cluster Centroids
